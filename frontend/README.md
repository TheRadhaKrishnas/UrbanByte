# UrbanByte Frontend Prototype

This is a frontend prototype for UrbanByte – a residential society management platform.

## Setup

1. Clone or download this folder.
2. Open the folder in VS Code.
3. Open `index.html` with Live Server or directly in browser.

## Features

- Login page with role selection (Member / Management)
- Member dashboard with sidebar and main content area
- Management dashboard with additional management features
- Profile, Complaint, Polls, Emergency, Payment, Documents pages
- Mock data and local storage for authentication simulation

## Notes

- This is a demo frontend only. No real backend.
- Use any credentials to login; mock users:
  - Member: MEM123 / pass
  - Management: ADMIN / admin
- All data is static or stored in sessionStorage temporarily.
